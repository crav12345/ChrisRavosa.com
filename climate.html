Climate Monitoring System

This dashboard visualizes real-time telemetry from an IoT climate monitoring system Chris built. An ESP32-based device runs fault-tolerant firmware that streams sensor data to a Node.js API deployed on Google Cloud. This webpage maintains a persistent WebSocket connection to receive low-latency, real-time updates. The system supports full bidirectional communication, enabling both live telemetry streaming and remote device control.

The code for the ESP32 firmware, Node.js server, and ChrisRavosa.com are all public on Chris's GitHub.
