HOME RUN DERBY

C# TCP Matchmaking Servers

I built and maintained TCP matchmaking servers in C# for Home Run Derby Mobile. The service supported the game's live player traffic and connected players with multiplayer sessions.

I also developed server stress-testing infrastructure that generated simulated client traffic, allowing the team to exercise the matchmaking backend under load and validate its behavior before deployment.

This work was part of the broader online platform supporting MLB's flagship live-service mobile game.
