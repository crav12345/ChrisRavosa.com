Real-Time Climate

This dashboard visualizes real-time telemetry from a custom-built, IoT climate monitor developed by Christopher. The IoT device and webpage communicate via websockets on a server hosted on Google Cloud Platform. The system is bidirectional, allowing remote control of the climate monitor from this dashboard.

The code for ChrisRavosa.com, the ESP32 firmware, and the websocket server is all available on Christopher's GitHub.
